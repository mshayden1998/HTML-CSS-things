Observações:

Perceba que o menu é chamado através do elemento iframe.

Nunca esqueça de acrescentar a propriedade target="_top" em todos os links (tag a) para que uma página não se abra apenas dentro do elemento iframe.

Funciona, mas use apenas para facilitar o desenvolvimento. De preferência não deixe na "versão final" do seu website.

Para o css externo funcionar no seu menu horizontal, você deve chamá-lo através da tag link pois se trata de outro documento HTML.
